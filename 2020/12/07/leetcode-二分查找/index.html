480. Sliding Window Median

给定一个数组num，存在一个大小为k的滑动窗口，该窗口从数组的最左边移到最右边。您只能在窗口中看到k个数字。每次滑动窗口向右移动一个位置。您的工作是输出原始数组中每个窗口的中值数组。

思路： 维持一个有序数组，例如第一次的窗口数组为[1,3,-1],我们可以把它先进行排序变成[-1,1,3]。

接下来对于要进来的数字，我们要进行的操作是，删除在window里原有nums数组的第一个数值，在window里插入nums数组中接下来的数字，同时保证数组有序。
这个部分我们可以用二分查找来做，删除操作时可以在logN的时间内找到，添加数字时，由于数组中没有这个数字，但是l,r的位置会锁定添加数字的位置。

时间复杂度为nlogN,暴力解法复杂度为N^2，因为每次都要进行排序;

74. Search a 2D Matrix

写一种有效的算法，以在m x n矩阵中搜索值。该矩阵具有以下属性： 每行中的整数从左到右排序。 每行的第一个整数大于前一行的最后一个整数。

二分搜索的典型题目，假设我们有m*n 的矩阵,有两种思路：

  1. 一种是我们可以从矩阵最右上角的值进行判断,row=0,col=matrix[0].length-1，matirx[row][col]==target, 大于的话， col–, 小于的话 row++.
  2. 第二个方法是把矩阵看成数组的形式，matrix ==> arr ===>matrix[mid/m]][mid%m], m为一行的长度，比如matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,50]]， 我们可以看成长度为12的数组，mid=Math.floor((l+r)/2) // ==>5, 那么中间值的位置为matrix[5/4][5%4] ==> matrix[1][1]
